# Calendar Conventions

## Calendars

| Calendar | Context | Description |
|----------|---------|-------------|
| Calendar | Work | Work meetings, 1:1s, presentations — from Exchange/Outlook |
| valencia.ag@gmail.com | Personal | Personal appointments and events |
| Hemma | Family | Shared family calendar with Sofia |

Other calendars (Holidays, Birthdays, etc.) are informational and can be shown but don't need special handling.

## Hemma (Family) Naming Convention

Events in the **Hemma** calendar follow a prefix convention to indicate ownership:

| Prefix | Meaning | Example |
|--------|---------|---------|
| `A: ...` | Agustín only — visible to Sofia for awareness | `A: Dentist appointment` |
| `S: ...` | Sofia only — visible to Agustín for awareness | `S: Yoga class` |
| *(no prefix)* | Both — shared family event | `Dinner at parents` |

When displaying or summarising Hemma events, interpret the prefix to provide context (e.g. "your dentist appointment" vs "Sofia's yoga class" vs shared events).
