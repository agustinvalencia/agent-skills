# Agent Skills Roadmap

Tracking skills for the ADHD-friendly productivity system.

## Implemented (18 skills)

### Daily Routines
| Skill | Version | Description |
|-------|---------|-------------|
| start-day | 3.0 | Morning briefing with standup - ONE task, ONE intention |
| close-day | 2.0 | Evening wind-down - intention reflection, acknowledge, capture, rest |

### Reviews & Reports
| Skill | Version | Description |
|-------|---------|-------------|
| weekly-review | 3.0 | Weekly reflection - wins first, ONE goal |
| monthly-report | 3.0 | Generate monthly summary - uses activity reports |
| project-review | 1.0 | Deep dive on single project health |

### Task Flow

## Design Principles

All skills follow [ADHD-PRINCIPLES.md](./references/ADHD-PRINCIPLES.md):
- One thing at a time
- Celebrate wins first
- Compassionate accountability
- Low friction
- External memory
